Sorting Algorithms 1: Selection Sort

Selection sort is a sorting algorithm that treats the input as two parts, a sorted part and an unsorted part and repeatedly selects the proper next value to move from the unsorted part to the end of the sorted part. 

        
            def selection_sort(numbers):
                for i in range(len(numbers) - 1):
                    index_lowest = i
                    for j in range(i + 1, len(numbers)):
                        if numbers[j] < numbers[index_lowest]:
                            index_lowest = j
                    temp = numbers[i]
                    numbers[i] = numbers[index_lowest]
                    numbers[index_lowest] = temp
                return numbers

Steps:

  • Input:

    a list of numbers ex: [64, 25, 12, 22, 11]

  • Variables:

    • numbers: This is the list of numbers that will be sorted.

    • i: This variable represents the index of the current element being considered as the smallest element in the unsorted part of the list. It starts from the first index (0) and moves towards the end of the list.

    • index_lowest: This variable keeps track of the index of the smallest number found in the unsorted portion of the list. It is initialized to the value of i at the beginning of each iteration.

    • j: This variable is used to iterate through the elements in the unsorted portion of the list, starting from the index i + 1 to the end of the list. It helps to find the smallest element.

    • temp: This temporary variable is used for swapping elements. It stores the value of the element at index i before it is replaced by the smallest element found (at index_lowest).

  • Sorting Process:

    1. For each element i from the beginning of the list to the second last element:

      • Initialize index_lowest to i.
      • Compare the element at index_lowest with each subsequent element (using j).
      • If a smaller element is found (i.e., numbers[j] < numbers[index_lowest]), update index_lowest to j.

    2. After the inner loop finishes, swap the smallest element found with the element at index i:

      • Store numbers[i] in temp.
      • Set numbers[i] to numbers[index_lowest].
      • Set numbers[index_lowest] to temp.
